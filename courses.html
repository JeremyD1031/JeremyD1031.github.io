Relevant Courses

Course Description
CSC 1710: Introduction to Computer Science Foundational course covering programming basics, problem-solving techniques, and computational thinking.
CSC 1720: Advanced Programming with Data Structures Continuation of introductory programming with focus on data structures such as arrays, linked lists, stacks, and queues.
CSC 2212: Database Systems Study of database design, SQL, relational models, and database management systems.
CSC 2342: Discrete Structures Mathematical foundations for computer science including logic, sets, relations, functions, and graph theory.
CSC 2410: Computer Systems Exploration of computer architecture, assembly language, memory management, and system-level programming.
CSC 2710: Advanced Data Structures In-depth study of complex data structures including trees, heaps, hash tables, and graphs with algorithm analysis.
CSC 3212: Web Technologies Development of web applications using HTML, CSS, JavaScript, and server-side technologies.
CSC 3810: Introduction to Information Security Overview of cybersecurity principles, threat analysis, cryptography, and security best practices.
CSC 3940: System Security Advanced security concepts including network security, vulnerability assessment, and secure system design.
MTH 1400: Pre-Calculus Preparation for calculus covering functions, trigonometry, and algebraic concepts.
MTH 1410: Calculus I Introduction to differential calculus including limits, derivatives, and their applications.
MTH 2310: Linear Algebra Study of vectors, matrices, linear transformations, and systems of linear equations.
STS 2910: Introduction to Statistical Analysis Fundamentals of statistics including probability, data analysis, and statistical inference.
